# gpt-in-terminal
---
# Installation
```
git clone https://github.com/ramazanix/gpt-in-terminal.git && \
cd gpt-in-terminal && \
./install.sh
```

# Usage
```
gpt "2+2=?"
```

# Uninstall
```
rm ~/.chat-gpt.py && \
python3 -m pip uninstall g4f && \
unalias gpt
```
**And remove alias from your ~/.(bash|zsh)rc file on the last line**
